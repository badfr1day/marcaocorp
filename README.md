# Lista de Tarefas

Projeto desenvolvido para a disciplina de Desenvolvimento de Sistemas.

## Objetivo

Aplicar na prática os conceitos de versionamento de código utilizando Git e GitHub.

## Funcionalidades

- Adicionar tarefas
- Marcar tarefa como concluída (clique na tarefa)
- Remover tarefa individualmente

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Como executar

1. Clone o repositório:
   git clone https://github.com/badfr1day/marcaocorp.git
2. Abra o arquivo index.html no navegador.

## Conceitos de Git utilizados

- Repositório no GitHub
- Commits organizados
- Branches para desenvolvimento de funcionalidades
- Issues para rastreamento de tarefas
- Pull Requests para merge das alterações
